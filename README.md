## 🎯 Overview

**Learn English** is a comprehensive web application that generates professional Grade 3 English lessons following the Tamil Nadu curriculum. Create:

✅ Complete lesson content (600-800 words)
✅ 15 differentiated exercises (3 difficulty levels)
✅ Vocabulary builder with 20+ key words
✅ Teacher notes and answer keys
✅ Tamil translations (optional)
✅ Professional Word/PDF documents

## 🚀 Quick Start

```bash
# Install
npm install

# Development
npm run dev

# Build
npm run build
```

Open [http://localhost:5173](http://localhost:5173)
